# Documentation

This directory contains system designs, flow charts, API documentation, and architecture diagrams for the AI-Powered Job Recommendation Assistant.

## Documents Catalog

- **System Architecture**: High-level structural overview of the React frontend, FastAPI backend, Qdrant database, and Groq integration.
- **API Spec**: Listing of endpoints, request/response formats, and token details.
- **RAG Workflow**: Breakdown of resume text extraction, semantic matching, and prompt engineering steps.
